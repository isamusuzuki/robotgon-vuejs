# robotgon-vuejs

ロボットゴン Vue.js 3 勉強

作成日 2021/09/19、更新日 2021/11/16

## 01. ファイル・フォルダ構成

```text
--robotgon-vuejs3/
    |--public/  ... コンパイル後のJavaScriptの置き場
    `--src/     ... コンパイル前のTypeScriptの置き場
```

## 02. Node.js 開発環境

```bash
# パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
# => ブラウザで http://localhost:8080 を開く

# JSコードのビルド
npm run build
```
